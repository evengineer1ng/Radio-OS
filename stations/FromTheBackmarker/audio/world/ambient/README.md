# Ambient Motorsport Audio

Optional atmospheric layers (future enhancement).

## Potential Assets

### Garage / Workshop
- Pneumatic tools
- Distant PA announcements
- Mechanic chatter
- Rolling toolboxes

### Paddock
- Tire warmers humming
- Generator drone
- Distant engines firing up
- Radio chatter

### Crowd / Trackside
- Crowd murmur
- Grandstand reactions
- Track announcer (indistinct)
- Air horns, flags waving

## Technical Requirements

- **Format**: OGG Vorbis (long loops)
- **Sample Rate**: 44.1 kHz
- **Duration**: 2-5 minutes loopable
- **Peak Level**: -12dB (background texture, not foreground)

## Usage

Not currently implemented in v1.0 audio engine. Reserved for future atmospheric enhancement.
